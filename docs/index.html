Who We Are

Perception · Planning · Control.

MACARON은 자율주행의 전 단계를 직접 개발하고, 실차 위에서 완성합니다. 특히 End-to-End(E2E) 학습을 핵심 연구 방향으로 삼아, 인지부터 판단, 제어까지를 하나의 통합된 시스템으로 구현하는 연구를 진행하고 있습니다.

Vision
System

라이다, 카메라 등 다양한 센서 데이터를 활용해 주변 환경을 정밀하게 인지하고, 자율주행 판단에 필요한 객체와 도로 정보를 안정적으로 추출합니다.

Planning
Logic

GPS와 IMU 기반 위치 추정 결과를 바탕으로, 차량이 주행해야 할 경로와 행동을 실시간으로 계획하여 안전하고 효율적인 주행을 만듭니다.

Hardware
Operation

차량 하드웨어와 센서 시스템을 직접 관리하고, 대회 운영과 문서화까지 담당해 실차 테스트와 현장 주행이 안정적으로 이어지도록 지원합니다.
